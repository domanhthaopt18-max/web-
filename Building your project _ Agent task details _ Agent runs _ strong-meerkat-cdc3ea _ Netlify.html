  • Home page (/) — Lists products from a static data catalog with alternating image/text layouts
  • Product detail page (/products/:productId) — Individual product pages
  • AI assistant — Infrastructure is in place for a chat assistant powered by Anthropic/OpenAI/Gemini (via environment variables)
